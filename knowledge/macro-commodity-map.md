# 宏观→品种映射知识库

> **核心原则**：不同的宏观变量对不同品种的影响方向和力度不同。本知识库是你分析宏观事件对特定品种影响的基础参考。
> 
> **使用方式**：每当出现宏观数据或事件时，先查此表，找到受影响的品种，再评估影响方向和程度。

## 二、逐品种宏观传导路径

### 🏗️ 玻璃 (FG) —— 竣工端品种，房地产最敏感

```
房地产销售↑ → 房企资金改善 → 竣工加速 → 玻璃需求↑ → FG↑
房地产新开工↑ → 领先2-3年 → 远期竣工预期 → FG预期↑
纯碱价格↑ → 玻璃成本↑ → 利润压缩 → 供给减少预期 → FG↑（成本支撑）
光伏装机↑ → 光伏玻璃需求↑ → 玻璃需求↑ → FG↑
社融↑ → 房地产资金面改善 → 竣工预期↑ → FG↑
LPR下调 → 房贷利率↓ → 地产销售↑ → 竣工预期↑ → FG↑
```

**关键数据响应优先级：**
1. 房屋竣工面积（同步指标，权重最高）
2. 商品房销售面积（领先6-12个月）
3. 社会融资规模（领先3-6个月）
4. 房地产开发投资
5. 房屋新开工面积（领先2-3年）
6. 纯碱价格（成本端）
7. 光伏新增装机（新需求）
8. LPR/房贷利率

**特殊属性：**
- 玻璃是房地产链条的"末端品种"，对竣工端政策（如保交楼）极其敏感
- 玻璃冷修成本高、周期长，供给调整慢，短期价格弹性大
- 季节性：春节后复工、金九银十是旺季

---

### 🧪 纯碱 (SA) —— 玻璃的上游，间接宏观品种

```
玻璃需求↑ → 玻璃厂开工↑ → 纯碱需求↑ → SA↑
光伏玻璃投产↑ → 纯碱需求↑ → SA↑
房地产竣工↑ → 玻璃需求↑ → 纯碱需求↑ → SA↑
环保限产 → 纯碱供给↓ → SA↑
煤炭/天然气↑ → 纯碱成本↑ → SA↑（成本支撑）
原盐价格↑ → 纯碱成本↑ → SA↑
新增产能投放 → 供给↑ → SA↓
```

**关键数据响应优先级：**
1. 玻璃生产线开工率（直接需求指标）
2. 纯碱库存（供需最直接反映）
3. 光伏玻璃在产产能
4. 房地产竣工面积（间接需求）
5. 煤炭/天然气价格（成本）
6. 纯碱开工率（供给端）

**特殊属性：**
- 纯碱的宏观传导是"二手的"——宏观先影响玻璃，玻璃再影响纯碱
- 但光伏需求让纯碱有了独立于房地产的逻辑
- 纯碱历史上波动极大，宏观解释力有限，供需基本面更重要
- 关注纯碱-玻璃价差（产业链利润分配）

---

### 🔩 螺纹钢 (RB) —— 投资端品种，对国内宏观最敏感

```
房地产新开工↑ → 螺纹钢需求↑ → RB↑
基建投资↑ → 螺纹钢需求↑ → RB↑
社融↑ → 投资资金到位 → 螺纹钢需求↑ → RB↑
专项债发行↑ → 基建资金到位 → 螺纹钢需求↑ → RB↑
PMI↑ → 制造业活动↑ → 钢材需求↑ → RB↑
铁矿石↑ → 成本↑ → RB↑（成本推动）
环保限产 → 供给↓ → RB↑
LPR下调 → 房贷利率↓ → 地产销售↑ → 新开工预期↑ → RB↑
```

**关键数据响应优先级：**
1. 房屋新开工面积（最直接的需求指标，权重最高）
2. 基建投资增速（第二需求支柱）
3. 社会融资规模（资金面，领先）
4. PMI（整体经济活动）
5. 商品房销售面积（领先新开工）
6. 专项债发行进度（基建资金）
7. 铁矿石/焦炭价格（成本端）
8. 环保限产政策（供给端）

**特殊属性：**
- 螺纹钢是中国经济的"体温计"，对中国投资周期最敏感
- 春节后复工行情（3-4月）是全年最重要的交易窗口
- 螺纹钢-热卷价差反映建筑业 vs 制造业相对强弱
- 季节性极强：雨季（6-8月）需求淡季，金九银十（9-10月）旺季

---

### 🏭 热卷 (HC) —— 制造业品种

```
制造业PMI↑ → 热卷需求↑ → HC↑
汽车产销↑ → 热卷需求↑ → HC↑
家电出口↑ → 热卷需求↑ → HC↑
出口订单↑ → 热卷需求↑ → HC↑
铁矿石↑ → 成本↑ → HC↑
海外PMI↑ → 出口需求↑ → HC↑
```

**关键数据响应优先级：**
1. 制造业PMI（新订单分项）
2. 汽车产销数据
3. 出口订单数据
4. 家电产销
5. 海外制造业PMI（出口需求）
6. 铁矿石价格（成本）

**特殊属性：**
- 卷螺差（HC-RB）是观察制造业vs建筑业相对强弱的核心指标
- 出口退税政策变化对热卷影响大
- 对海外宏观更敏感（出口占比高）

---

### ⛏️ 铁矿石 (I) —— 中国需求 + 全球供给

```
中国房地产新开工预期↑ → 钢厂补库 → 铁矿石需求↑ → I↑
中国PMI↑ → 钢材需求预期↑ → 铁矿石需求↑ → I↑
美元↓ → 大宗商品计价↓ → I↑
澳洲/巴西发货量↓ → 供给↓ → I↑
BDI↑ → 运输成本↑ → I↑
钢厂利润↑ → 补库意愿↑ → I↑
中国粗钢产量↓ → 铁矿石需求↓ → I↓（利空）
```

**关键数据响应优先级：**
1. 中国房地产新开工/销售（需求预期）
2. 四大矿山发货量（供给端）
3. 中国港口铁矿石库存（供需平衡）
4. 美元指数（计价因素+全球风险偏好）
5. 中国粗钢产量（直接需求）
6. BDI/西澳运费（运输成本）
7. 钢厂利润率（补库意愿）

**特殊属性：**
- 铁矿石是中国定价 vs 矿山定价的博弈，宏观影响通过"预期"传导
- 四大矿山（力拓、必和必拓、FMG、淡水河谷）高度垄断，供给端事件影响大
- 天气因素（澳洲飓风、巴西雨季）对短期供给冲击大
- 中国粗钢产量压减政策是独特的政策风险

---

### 🔥 焦炭 (J) —— 夹在煤和钢之间

```
钢厂开工率↑ → 焦炭需求↑ → J↑
焦煤价格↑ → 焦炭成本↑ → J↑
环保限产 → 焦炭供给↓ → J↑
房地产预期↑ → 钢材需求预期↑ → 焦炭需求↑ → J↑
吨钢利润↑ → 钢厂接受焦炭涨价能力↑ → J↑
```

**关键数据响应优先级：**
1. 高炉开工率（直接需求）
2. 焦煤价格（成本端，占比最大）
3. 吨钢利润（需求容忍度）
4. 环保限产政策（供给端）
5. 焦化厂开工率

**特殊属性：**
- 焦炭宏观敏感度较低，更多受产业自身供需驱动
- 但焦炭是"夹心层"——涨跌都受限于上下游
- 环保限产是焦炭最大的独立宏观/政策变量

---

### 🪨 焦煤 (JM) —— 供给侧故事最多的品种

```
焦炭需求↑ → 焦煤需求↑ → JM↑
煤矿安全生产检查 → 供给↓ → JM↑
进口政策变化（蒙古/澳洲） → 供给端冲击 → JM波动↑
中澳关系 → 澳煤进口预期 → JM↓（澳煤恢复进口利空）
蒙古通关量 → 蒙煤供给 → JM↓（通关增加利空）
```

**关键数据响应优先级：**
1. 焦化厂开工率（需求）
2. 煤矿安全检查/环保政策（供给）
3. 进口通关政策
4. 蒙古煤炭通关量
5. 钢厂利润（通过焦炭传导）

**特殊属性：**
- 焦煤供给端的故事（安全、环保、进口）常常比需求端更重要
- 政策影响极大——中澳关系、蒙古口岸政策直接决定供给
- 宏观敏感度在黑色系中最低

---

### 🔶 铜 (CU) —— "铜博士"，全球宏观风向标

```
全球PMI↑ → 铜需求↑ → CU↑
中国PMI↑ → 铜需求↑（占全球50%） → CU↑
美元↓ → 铜价↑（负相关极强） → CU↑
美债收益率↓ → 持有成本↓ → 铜价↑
中国社融↑ → 投资活动↑ → 铜需求↑ → CU↑
新能源投资↑ → 铜需求↑（电动车/光伏/风电） → CU↑
LME库存↓ → 供给偏紧 → CU↑
美国利率↓ → 美元↓ + 经济预期↑ → CU↑
全球经济衰退预期 → 铜需求↓ → CU↓
中国房地产↓ → 铜需求↓（建筑用铜） → CU↓
```

**关键数据响应优先级：**
1. 全球/中国PMI（同步指标，权重最高）
2. 美元指数（负相关，日度可观察）
3. 美国10年期国债收益率（通过美元和风险偏好）
4. 中国社融/信贷（领先2-3个月）
5. 中国房地产数据（建筑用铜占比较大）
6. LME+SHFE铜库存（供需平衡表）
7. 美国利率预期（CME FedWatch）
8. 全球新能源汽车销量（新需求）

**特殊属性：**
- 铜被称为"铜博士"，是预测全球经济最好的商品指标之一
- 铜是宏观交易者最青睐的商品，宏观经济数据对铜的影响往往快且大
- 铜受"全球风险偏好"影响极大，有时与基本面脱钩
- 关注铜金比（铜价/金价）——反映市场对经济增长vs避险的偏好
- "铜就是另一种形式的PMI"——两者相关性极高

---

### 🔧 铝 (AL) —— 供给约束 + 需求多元

```
中国PMI↑ → 铝需求↑ → AL↑
美元↓ → 铝价↑ → AL↑
基建投资↑ → 铝需求↑（建筑/电力） → AL↑
汽车产销↑ → 铝需求↑ → AL↑
电力成本↑ → 铝成本↑ → AL↑（中国电解铝用电占比30-40%）
中国产能上限（4500万吨）→ 供给受限 → AL长期有支撑
云南水电季节性 → 供给波动 → AL短期波动
```

**关键数据响应优先级：**
1. 中国PMI（需求端）
2. 美元指数（计价）
3. 中国基建/电力投资
4. 汽车产销
5. 电力价格（成本端）
6. LME铝库存
7. 中国电解铝运行产能

**特殊属性：**
- 铝的供给端有"中国4500万吨产能天花板"——这是政策硬约束
- 云南水电对供给的季节性影响（枯水期限电）
- 对比铜，铝的宏观敏感度低一些，但供给端故事更多

---

### 🪙 锌 (ZN) —— 镀锌需求 = 基建 + 地产竣工

```
基建投资↑ → 镀锌需求↑（电力铁塔/交通设施） → ZN↑
房地产竣工↑ → 镀锌需求↑（建筑用镀锌板/管） → ZN↑
汽车产销↑ → 镀锌需求↑ → ZN↑
美元↓ → 锌价↑ → ZN↑
全球PMI↑ → 锌需求↑ → ZN↑
```

**关键数据响应优先级：**
1. 基建投资增速（镀锌最大需求端）
2. 房地产竣工面积
3. 汽车产销
4. 美元指数
5. LME锌库存

**特殊属性：**
- 锌是"基建+地产竣工"的双重受益品种
- 锌与螺纹钢在需求端有一定重叠（都受益于基建）
- 锌矿供给（海外矿山）也是重要变量

## 五、使用本知识库的要点

1. **不是每个宏观事件都需要分析所有品种**——先看事件类型，再看对照表
2. **权重很重要**——社融对螺纹钢是五颗星的影响，对纯碱只有三颗星
3. **时效性**——有些影响是即时的（PMI公布后铜价瞬间反应），有些是渐进的（房地产政策见效需要数月）
4. **预期差是关键**——市场已经 price-in 的预期不会引起波动，超预期/不及预期才会
5. **多因素叠加**——单一宏观事件很少能独自驱动行情，要综合判断宏观面"合力"方向
6. **配合技术面**——宏观分析告诉你方向，盘面告诉你时机，二者缺一不可
