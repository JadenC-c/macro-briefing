# 宏观指标解读指南

> **目的**：帮你理解每个宏观指标的含义、如何解读、对期货品种的传导路径。

---

## 一、经济增长类指标

### 1. GDP（国内生产总值）
- **含义**：一个国家在一定时期内生产的所有最终商品和服务的总价值
- **频率**：季度
- **解读**：
  - GDP增速 > 预期 → 经济过热风险 → 央行可能收紧 → 短期利多商品（需求好），中期可能利空（政策收紧）
  - GDP增速 < 预期 → 经济放缓 → 央行可能放松 → 短期利空商品（需求差），中期可能利多（政策放松）
- **对品种影响**：全面影响，但频率低（季度），日常交易中权重不如月度指标

### 2. PMI（采购经理人指数）
- **含义**：通过对企业采购经理的调查，反映制造业/服务业的扩张或收缩
- **频率**：月度
- **关键阈值**：50 = 荣枯线，>50 扩张，<50 收缩
- **中国两个PMI的区别**：
  - **官方PMI**（国家统计局）：样本以大型国企为主，更反映重工业和基建
  - **财新PMI**（财新/Markit）：样本以中小民企为主，更反映出口和轻工业
- **解读**：
  - PMI > 预期 → 经济活动扩张 → 工业品需求↑ → 利多铜、螺纹、热卷
  - PMI < 预期且 < 50 → 经济收缩信号 → 工业品需求↓ → 利空
  - 关注分项：新订单（领先）、生产（同步）、原材料购进价格（通胀预期）
- **品种敏感度**：铜 > 螺纹钢 > 热卷 > 铁矿石 > 铝 > 锌 > 焦炭 > 玻璃 > 纯碱
- **公布时间**：中国官方PMI每月最后一天9:30，财新PMI每月第一个工作日9:45

### 3. 工业增加值
- **含义**：工业企业生产活动新创造的价值，反映工业生产的实际增长
- **频率**：月度
- **解读**：
  - 增速 > 预期 → 工业生产旺盛 → 利多工业品（铜、铝、螺纹）
  - 持续下滑 → 工业需求萎缩 → 利空
- **品种敏感度**：铜（电力设备）、热卷（制造业）、螺纹钢

### 4. 固定资产投资
- **含义**：建造和购置固定资产的工作量，含房地产、基建、制造业三大分项
- **频率**：月度（累计同比）
- **解读**：
  - 增速↑ → 建筑需求↑ → 利多螺纹钢、铁矿石、锌
  - 三大分项中，**基建投资**和**房地产投资**对商品影响最大
- **品种敏感度**：螺纹钢 > 铁矿石 > 焦炭 > 锌 > 铜

## 二、房地产类指标

### 5. 房屋新开工面积
- **含义**：报告期内新开工建设的房屋建筑面积
- **频率**：月度（累计同比）
- **解读**：
  - 这是**螺纹钢需求最直接的前瞻指标**（盖房子打地基、做结构大量用钢筋）
  - 新开工↑ → 螺纹钢未来3-6个月需求↑ → 利多RB；远期利多玻璃（2-3年后竣工）
  - 新开工连续下滑 → 黑色系需求萎缩
- **品种敏感度**：螺纹钢（最敏感）> 铁矿石 > 焦炭 > 玻璃（远期预期）> 铜
- **公布时间**：每月15-18日，国家统计局

### 6. 房屋竣工面积
- **含义**：报告期内房屋建筑按照设计要求已全部完工、达到使用条件
- **频率**：月度（累计同比）
- **解读**：
  - 这是**玻璃需求最直接的指标**（竣工时才安装门窗、幕墙玻璃）
  - 竣工↑ → 玻璃需求↑ → 利多FG
  - 保交楼政策的核心效果指标
- **品种敏感度**：玻璃（最敏感）> 纯碱（通过玻璃传导）> 锌（镀锌管/板）
- **公布时间**：每月15-18日，国家统计局

### 7. 商品房销售面积
- **含义**：报告期内出售的商品房建筑面积
- **频率**：月度（累计同比）
- **解读**：
  - 销售是最领先的指标——销售回暖→房企回款→拿地→新开工→施工→竣工
  - 销售↑ → 整个地产链条预期改善 → 利多黑色系+建材
- **品种敏感度**：螺纹钢 > 玻璃 > 铁矿石 > 纯碱
- **公布时间**：每月15-18日，国家统计局

## 三、货币信贷类指标

### 8. 社会融资规模（社融）
- **含义**：一定时期内实体经济从金融体系获得的资金总额，含贷款、债券、股票、信托等
- **频率**：月度
- **解读**：
  - 社融是中国经济最重要的领先指标之一（领先经济约2-3个月）
  - 社融↑ → 资金进入实体经济 → 投资活动↑ → 利多黑色系、有色
  - 关注结构：**居民中长期贷款**（对应房贷）直接关系房地产
  - 关注结构：**企业中长期贷款**对应企业投资扩产
- **品种敏感度**：螺纹钢 > 铜 > 玻璃 > 铁矿石 > 铝 > 锌
- **公布时间**：每月9-15日，盘中不定期公布

### 9. M2（广义货币供应量）
- **含义**：流通中的现金 + 活期存款 + 定期存款 + 储蓄存款
- **频率**：月度
- **解读**：
  - M2增速↑ → 市场流动性充裕 → 通胀预期↑ → 利多商品
  - 但要结合实体需求看：如果M2高但社融低 = 钱在金融体系空转，对商品帮助有限
- **品种敏感度**：铜 > 螺纹钢 > 铁矿石

### 10. LPR（贷款市场报价利率）
- **含义**：银行对最优质客户的贷款利率，是实际贷款利率的基准
- **频率**：每月20日公布
- **解读**：
  - LPR下调 → 贷款成本↓ → 利好房地产和投资 → 利多黑色系+玻璃
  - 5年期LPR（主要对应房贷）比1年期LPR对商品影响更大
- **品种敏感度**：螺纹钢 > 玻璃 > 铜 > 铁矿石

### 11. MLF（中期借贷便利）
- **含义**：央行向商业银行提供中期资金的货币政策工具
- **解读**：
  - MLF利率下调 → LPR下调的前兆 → 利多
  - MLF超额续作 → 央行向市场注入流动性 → 利多
- **品种敏感度**：与LPR类似

## 四、通胀类指标

### 12. CPI（消费者物价指数）
- **含义**：反映居民家庭购买消费品和服务项目的价格变动
- **频率**：月度
- **解读**：
  - CPI持续走高 → 央行可能收紧货币 → 利空商品
  - CPI过低（通缩） → 央行可能放松 → 利多商品
  - 中国CPI中猪肉权重高，要剔除食品看核心CPI
- **品种敏感度**：间接影响，主要通过政策预期传导

### 13. PPI（工业生产者出厂价格指数）
- **含义**：工业企业产品出厂价格的变动趋势
- **频率**：月度
- **解读**：
  - PPI↑ → 工业品价格上涨 → 上游企业利润改善 → 可能利多商品
  - PPI负值 → 工业通缩 → 需求不足 → 利空工业品
- **品种敏感度**：螺纹钢、铜、铁矿石（PPI与工业品价格互为因果）

## 五、海外关键指标

### 14. 美国非农就业
- **含义**：美国非农业部门新增就业人数
- **频率**：月度
- **为什么重要**：美联储决策最重要的参考指标之一
- **解读**：
  - 非农大幅超预期 → 经济强劲 → 美联储可能不急于降息 → 美元↑ → 铜/有色↓
  - 非农大幅低于预期 → 经济走弱 → 降息预期↑ → 美元↓ → 铜/有色↑
  - **注意**：有时候"好消息是坏消息"（数据好→加息预期→利空商品），有时候"坏消息是好消息"（数据差→降息预期→利多商品），取决于当前市场更关注增长还是通胀
- **品种敏感度**：铜 > 铁矿石 > 铝 > 锌
- **公布时间**：每月第一个周五 20:30（北京时间）

### 15. 美国CPI
- **含义**：美国消费者物价指数
- **频率**：月度
- **为什么重要**：决定美联储利率路径
- **解读**：
  - CPI高于预期 → 美联储可能加/维持高利率 → 美元↑ → 铜/有色↓
  - CPI低于预期 → 降息预期↑ → 美元↓ → 铜/有色↑
- **品种敏感度**：铜最敏感，其次铁矿石、铝、锌
- **公布时间**：每月10-14日 20:30

### 16. 美联储利率决议（FOMC）
- **含义**：美联储联邦公开市场委员会对利率的决定
- **频率**：每6周一次
- **为什么重要**：直接影响全球资产定价的锚（无风险利率）
- **解读**：
  - 降息 → 美元↓ + 风险偏好↑ → 大利多铜、铁矿石、有色
  - 加息 → 美元↑ + 风险偏好↓ → 利空
  - 维持不变但释放鸽派信号 → 美元↓ → 利多
  - 维持不变但释放鹰派信号 → 美元↑ → 利空
- **关键不是决议本身，是点阵图和鲍威尔发布会**
- **品种敏感度**：铜 > 铁矿石 > 铝 > 锌 > 螺纹钢（间接）
- **公布时间**：决议日凌晨2:00（北京时间），发布会2:30

### 17. 美元指数 (DXY)
- **含义**：美元对一篮子货币（欧元、日元、英镑等）的汇率
- **频率**：实时
- **为什么重要**：大宗商品以美元计价，美元涨则商品跌（但非绝对）
- **解读**：
  - 美元↑ → 商品↓（负相关，尤其是铜、铁矿石）
  - 美元↓ → 商品↑
  - **但是**：如果美元上涨是因为全球经济强劲（而非避险），商品可能同涨
- **品种敏感度**：铜（最强负相关）> 铁矿石 > 铝 > 锌 > 其他
- **对中国品种的影响**：主要通过（1）进口成本；（2）风险偏好；（3）汇率传导至国内品种

### 18. 美国10年期国债收益率
- **含义**：全球资产定价的基准
- **频率**：实时
- **解读**：
  - 收益率↑ → 债券↓ + 美元可能↑ → 商品承压
  - 收益率↓ → 持有商品的机会成本↓ → 利多商品
  - 关注**实际利率**（名义利率 - 通胀预期）：实际利率↓ → 黄金、铜利多
- **品种敏感度**：铜、黄金最敏感

## 六、解读技巧

### 预期差才是核心
市场从来不是对"数据好坏"做反应，而是对"数据与预期的差异"做反应。一个"好但不够好"的数据可能利空，一个"坏但没那么坏"的数据可能利多。
> 例：市场预期非农+20万，公布+18万 → 不及预期 → 降息预期↑ → 美元↓ → 利多铜
> 例：市场预期非农+5万，公布+8万 → "超预期"但绝对值差 → 可能短期利多但持续性差

### 分清"领先"和"滞后"
- PMI、社融、房地产销售 → 领先指标（提前2-6个月）
- GDP、工业增加值、房地产开工 → 同步指标
- CPI、失业率 → 滞后指标（落后经济拐点3-6个月）

### 关注数据修正
很多数据第一次公布后会修正。有时"修正值"比"初值"更重要——它能确认或推翻趋势。

### 组合判断
单一指标不足以判断方向，要关注指标间的互相印证：
- PMI↑ + 社融↑ + 地产销售↑ = 中国需求改善确认 → 坚定做多黑色系
- PMI↑ 但社融↓ = 数据可能有噪声，谨慎
