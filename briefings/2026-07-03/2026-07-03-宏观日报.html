3. 重大事件跟踪

【返回目录】

持续追踪影响品种的宏观事件，每日更新最新进展。G7峰会（6/17闭幕）已消退，本期移除。
持续 美联储政策路径：鹰派暂停→非农后加息预期骤降

事件：6/18 FOMC 沃什首秀维持利率 3.50-3.75% 不变，但删除前瞻指引、拒绝点阵图、设立五大工作组，释放强烈鹰派信号——年内加息概率一度飙至83%。

最新（7月2日）：美国6月非农仅新增 5.7万人（预期11万），4-5月合计下修7.4万人。加息预期骤降——市场将首次加息时点从10月推迟至12月，年内加息概率从83%降至约55%。但失业率降至4.2%（劳动参与率下降0.3pct，有统计"水分"），沃什重申2%通胀目标不动摇。

现状：非农爆冷暂时压制加息预期，但通胀（5月CPI 4.2%、PCE预测上调至3.6%）仍在高位。美联储进入"数据依赖"模式，下一个关键节点是7/14的6月CPI。市场定价在"加息"与"维持"之间摇摆，美元短期偏弱。

影响品种：铜（最敏感）→ 加息预期推迟 → 美元走弱 → 短期利多；铁矿石、铝、锌 → 间接利多，但"增长担忧"升温限制上行空间。
持续 美伊冲突与霍尔木兹海峡危机：多哈会谈无突破

事件：6/15 美伊签署谅解备忘录（MOU）宣布停火，但以色列持续搅局。60天过渡期处理核问题和制裁解除，核心博弈转移至霍尔木兹海峡通航权。

最新（7月2日）：① 7/1 美伊在多哈通过卡塔尔/巴基斯坦斡旋举行间接技术性谈判——三大议题（海峡通航、黎巴嫩停火、解冻资产）分歧严重，未取得突破。伊朗要求掌握海峡管理权，美方提出"解冻资产换海峡控制权"被拒。② 7/2 联合国安理会紧急公开会，美伊激烈互相指责——美方称伊朗"阻碍所有船只通过海峡"，伊方反驳"美以军事行动才是危机根源"。中方提出三点主张（停火/恢复通航/改善地区关系）。

现状：停火脆弱但勉强维持。海峡已出现三条不同航线，伊朗警告未按批准航线航行将面临"果断回应"。副总统万斯称"非必要不重返战场"，但以色列国防部长称已选定伊朗境内打击目标。60天窗口期内（截至8月中旬）解决所有分歧并不现实。

影响品种：全品种 → 地缘风险溢价反复 → 油价波动传导至整体商品情绪。若协议维持 → 风险溢价消退利多有色/黑色；若谈判破裂 → 油价报复性反弹重新利空。
持续 美国232铜关税调查：6/30报告确认15%分阶段征税

事件：2025年2月美国启动铜全品类232国家安全调查。2025年7月初步方案：铜半成品50%关税、精炼铜暂获豁免。

最新（6月30日）：商务部提交最终更新报告——再次确认精炼铜 15%关税（2027/1/1生效）→ 永久30%（2028/1/1生效）分阶段征税方案。无自动国别豁免（加/墨需总统单独审查），2027年起对美工业消费者实行年度国内铜消费配额。目前进入90天总统决策窗口期。

现状：基准情景（概率70%）：15%关税2027年如期落地。COMEX铜库存已飙升至65万吨历史新高（18个月抢运超120万吨），LME库存降至35万吨附近。高盛预测关税生效后铜价或突破$14,000。

影响品种：铜 → COMEX-LME价差持续走阔 → 伦铜短期承压但中期供给紧张支撑。铝 → 关注是否跟随加码。
持续 中国房地产深度调整 + 7月新政集中落地窗口期

事件：1-5月房地产数据全线恶化——开发投资-16.2%、新开工-22.6%、竣工-23.4%、销售-10.8%。"政策底"与"需求底"之间的煎熬持续。

最新（7月2日）：六大部委（住建部/央行/财政部/金融监管总局/税务总局/自然资源部）敲定 7月作为新政集中落地窗口期。核心举措：① 央行3000亿保障性住房专项再贷款全额启用、国资批量收储存量房源；② 首套房首付下调+房贷利率探底+"卖旧买新"税费减免；③ 去化周期超36个月城市暂停新增宅地供应；④ 超20城推进现房销售改革（广州为首个一线城市试点）。

现状：政策端全面加码——供给端（收储）、需求端（降首付/降利率）、土地端（暂停供地）、融资端（白名单应贷尽贷）四管齐下。但传导到实际需求需6-12个月，短期难以扭转新开工和竣工颓势。6月政治局会议定调"消化存量和优化增量相结合"。

影响品种：玻璃 → 保交房+现房销售 → 竣工端中期利多，但短期竣工仍-23.4%承压；螺纹钢 → 收储+暂停供地 → 远期新开工预期改善，但短期新开工-22.6%需求塌陷；纯碱 → 通过玻璃间接传导，光伏玻璃提供独立支撑。
持续 中国信贷收缩 + 内需疲弱：制造业边际改善，建筑业仍拖累

事件：5月社融2.03万亿超预期，但信贷结构严重失衡——票据融资占企业贷款超100%（5,570亿），居民中长期贷款减少571亿（房贷需求冻结）。

最新（7月1日）：6月官方制造业PMI回升至50.3%（时隔一个月重回扩张），新订单51.2%（+1.3pct）、新出口订单50.1%（+1.5pct），产需同步改善。但建筑业PMI仅49.0%仍在收缩。财新制造业PMI 51.7（连续第七个月扩张）、服务业PMI 54.1（高位小幅回落）。

现状：制造业边际改善但建筑业拖累仍在，"供强需弱"矛盾边际缓解但远未解决。6月社融数据（7月中旬公布）是检验信贷结构是否改善的关键节点。社零转负（5月-0.6%）是消费塌陷的警示信号。

影响品种：螺纹钢、玻璃 → 信贷结构恶化压制投资需求 → 短期利空；铜 → PMI回升+社融超预期 → 边际利多；铁矿石 → 建筑业PMI收缩+钢厂亏损 → 短期利空。
持续 中国城市更新十五五规划：2570亿资金启动

事件：6月国务院印发首部国家级城市更新专项规划，目标：老旧小区11.5万个、城中村4000个、管网77万公里，总投资≥15万亿（五年）。

最新（7月1日）：2026年中央预算内投资 970亿元 + 超长期特别国债 1600亿元（同比+250亿），合计2570亿元已下达。7月5日将在北京召开高规格城市更新专题会议（讨论投融管退等核心问题）。全国已有4省20城出台地方城市更新条例。

现状：资金端已开始落地，但项目端从规划→招标→开工仍需时间（预计Q3末-Q4进入施工高峰）。管网改造（77万公里）是螺纹钢/热卷最直接的需求增量；老旧小区改造利多远期的玻璃/锌需求。

影响品种：螺纹钢 → 管网改造直接拉动建筑钢材需求 → 中期利多；热卷 → 管材制造需求 → 中期利多；玻璃 → 老旧小区改造（门窗更换等）→ 远期利多；锌 → 镀锌管/板需求 → 中期利多。

5.1 建材板块（玻璃 FG · 纯碱 SA）

宏观面判断： 短期 中性偏空 · 中期 中性偏多

  1. 房地产竣工-23.4%压制玻璃需求 —— 1-5月竣工面积降幅仅微收窄0.6pct，短期玻璃需求基本面仍弱。但7月六大部委新政集中落地（3000亿收储全额启用+超20城现房销售），政策信号明确利多中期竣工端。
  2. 城市更新2570亿资金启动 —— 老旧小区改造（门窗更换）远期利多玻璃。但项目从规划到开工需时间，短期不会转化为实际需求。
  3. 纯碱-玻璃产业链利润分配 —— 纯碱涨+1.10%（1103元/吨），玻璃微涨+0.21%（966元/吨）。纯碱强于玻璃意味着玻璃利润压缩，但短期暂无冷修压力。
  4. 光伏玻璃提供独立支撑 —— 新能源政策持续加码，光伏新增装机为纯碱和玻璃提供独立于房地产的需求增量。
  5. 非农爆冷→美元走弱 —— 间接传导有限（建材以内需为主），但整体商品情绪偏正面。

盘面：FG2609 收 966元/吨（+0.21%），SA2609 收 1,103元/吨（+1.10%），纯碱资金净流入3.45亿居前。玻璃在950-980区间窄幅震荡，市场等待7月基本面变量。

本周关注：7/5 北京城市更新专题会议是否释放超预期投资信号；7月中旬6月房地产数据能否出现拐点。

5.2 黑色系（螺纹钢 RB · 热卷 HC · 铁矿石 I · 焦炭 J · 焦煤 JM）

宏观面判断： 短期 中性偏空 · 中期 中性

  1. 房地产新开工-22.6% + 建筑业PMI仅49.0% —— 螺纹钢最直接的两大需求指标同时收缩，7月雨季（传统淡季）叠加需求疲弱，短期基本面压力最大。
  2. 铁矿石库存1.67亿吨历史高位 + CMRG限购Fortescue —— 港口库存充足、钢厂亏损面扩大、铁水产量维持243万吨但盈利率下滑→负反馈风险上升。CMRG 7/15起限购超特粉（港口库存722万吨）短期加剧供给压力。
  3. 城市更新+专项债→基建需求托底 —— 管网改造77万公里目标直接利好螺纹钢/热卷。但专项债发行进度仍需跟踪，短期资金到位有限。
  4. 非农爆冷→美元走弱 —— 铁矿石以美元计价，美元走弱边际利多。但铁矿石自身基本面（高库存+钢厂亏损）限制上行空间。
  5. 热卷相对强于螺纹 —— 制造业PMI回升至50.3%+财新PMI 51.7连续扩张→制造业用钢需求好于建筑用钢。卷螺差可能走阔。

盘面：RB2610 收 3,067元/吨（-0.20%），I2609 收 740元/吨（+0.48%但7/3日内跌1.74%至734）。螺纹钢在3000-3100区间弱势震荡，铁矿石740关口承压，黑色系整体受制于需求疲弱+库存高位。

本周关注：铁水产量是否拐头（钢厂亏损减产→原料需求下滑→负反馈）；铁矿石港口库存趋势；7月中旬6月房地产+社融数据。

5.3 有色板块（铜 CU · 铝 AL · 锌 ZN）

宏观面判断： 短期 偏多 · 中期 中性偏多

  1. 非农爆冷 → 加息预期推迟 → 美元走弱 → 利多铜 —— 这是当前有色板块最核心的宏观逻辑。市场将加息时点从10月推迟至12月，美元指数从102+回落至101.41（盘中最低100.85），铜作为对美元最敏感的品种直接受益。
  2. 全球PMI信号偏积极 —— 中国官方PMI重回扩张（50.3）、财新连续七月扩张（51.7）→铜需求端（中国占全球50%）边际改善。欧元区服务业PMI终值今日公布也值得关注。但需警惕"增长担忧"——非农爆冷可能反映美国经济动能减弱。
  3. 232铜关税中长期利空伦铜 —— 6/30报告确认15%关税（2027年起），COMEX库存飙至65万吨。COMEX-LME价差已提前反映关税预期，伦铜短期承压但LME库存连降10日（约32.5万吨）提供支撑。
  4. 铝受电力成本+供给约束支撑 —— 中国4500万吨产能天花板+云南水电季节性波动提供供给端支撑，城市更新+基建利多铝需求。
  5. 锌受益于基建+城市更新 —— 管网改造77万公里→镀锌管/板需求，锌是基建最敏感的有色品种。

盘面：CU2608 收 101,950元/吨（-0.49%），伦铜$13,330.50（-0.33%）。沪铜在10-10.4万区间高位震荡，受美元走弱+关税不确定性双向拉扯。LME铝$3,092.50（+0.23%），LME锌$3,492.00（-1.68%）。

本周关注：今日欧洲服务业PMI终值+拉加德讲话；美元指数能否守住101关口；7/14美国6月CPI（决定加息预期能否进一步消退）。
