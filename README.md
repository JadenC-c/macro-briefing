# 宏观分析 Agent —— 期货交易员专属宏观助手

## 🎯 这个工具能做什么

我是一个 AI 驱动的宏观分析助手，专为你（期货交易员）服务。我能帮你：

1. **每天早上自动生成一份宏观盘前简报**——在你开盘前告诉你"今天宏观面对你的品种是偏多还是偏空"
2. **随时回答你的宏观问题**——比如"今天纯碱为什么跌？""非农对铜有什么影响？"

---

## 🚀 怎么用（零代码，一键操作）

### 方式一：生成每日宏观简报

**最简单的方法**：在 VSCode 中打开 Claude 对话框，输入：

```
帮我生成今天的宏观简报
```

Claude 会自动：
1. 去网上搜索最新的宏观数据、隔夜外盘行情、重要新闻
2. 对照知识库分析对你的品种（玻璃/纯碱/黑色/有色）的影响
3. 生成一份结构化的简报，保存到 `briefings/` 目录
4. 在聊天中展示给你看

**查看历史简报**：打开 `briefings/` 文件夹，按日期查看。

### 方式二：随时问宏观问题

直接在 Claude 对话框中输入你的问题，比如：

- "今天 PPI 数据公布了，对螺纹钢有什么影响？"
- "为什么今天铜跌了这么多？是不是宏观因素？"
- "美联储刚刚加息了，我的黑色系多单要不要先出来？"
- "最近房地产数据怎么样？对玻璃有没有利好？"
- "美元最近走强是什么原因？会影响铁矿石吗？"

Claude 会结合最新数据和知识库给你分析。

### 方式三：设置自动推送（推荐！）

让 AI 每个交易日早上自动生成简报：

在 Claude 对话框中输入：

```
帮我设置每天早上 7:30 自动生成宏观简报
```

Claude 会帮你设置好定时任务。之后每天早上你打开 VSCode，简报就已经生成好了。

## 🔧 个性化配置（可选）

如果你想调整关注品种或数据源，打开 `config.json`，修改 `trader.products` 部分：

```json
"products": [
  {"name": "玻璃", "code": "FG"},
  {"name": "纯碱", "code": "SA"},
  // ... 可以增减品种
]
```

如果某个品种你不做了，删掉那一行即可。新增品种同理。

## 📊 数据源说明

所有数据均来自免费公开网站：

| 数据 | 来源 |
|------|------|
| 中国经济数据 | 国家统计局、东方财富 |
| 全球数据日历 | ForexFactory |
| 隔夜外盘行情 | 新浪财经 |
| 宏观新闻 | 华尔街见闻、财联社、金十数据 |
| 品种基本面 | 东方财富期货 |

数据采集中如遇某些网站暂时不可访问，简报表会标注"数据暂不可得"，不影响其他部分。

## ⚠️ 重要提醒

1. **AI 分析仅供参考，不构成投资建议**。宏观分析告诉你"风吹向哪里"，但具体买卖点位需要结合你的盘面经验。
2. **宏观只是影响价格的一个维度**。品种自身的供需、库存、产业政策同样重要。简报表会综合判断，但不会替代你的基本面分析。
3. **数据有时滞**。部分宏观数据（如社融、房地产）每月只公布一次，两次公布之间盘面可能已提前反应预期。
4. **AI 可能出错**。如果引用的数据或逻辑让你觉得不对，多问一句，让 AI 重新核实。

## 🤔 常见问题

**Q：定时任务会自动一直运行吗？**
A：定时任务有效期7天，到期后会自动停止。每周对 Claude 说一次"帮我重新设置自动简报定时任务"即可续期。

**Q：如果某天没有生成简报怎么办？**
A：可能原因：1) 非交易日（周末/节假日）；2) 数据源暂时不可用；3) 定时任务到期未续；4) VSCode/Claude 未运行。你随时可以手动输入"生成今天的宏观简报"来触发。

**Q：简报内容不够详细怎么办？**
A：对任何一条内容，你可以追问。比如"展开说下非农对铁矿石的影响"，AI 会深入分析。

**Q：能同时看一个品种的宏观面和基本面吗？**
A：可以。例如输入"帮我分析一下玻璃——从宏观面和基本面两个角度"，AI 会做综合分析。

**Q：数据出错了怎么办？**
A：告诉 AI"这个数据好像不对，帮我重新核实一下"，AI 会重新搜索。
